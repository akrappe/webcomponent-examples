Web Component styling inside and outside of the Shadow DOM

Globally styled headline

The H2 and H3 headlines above are affected by the global styles defined in the head section.
The Web Components' H2 headline is affected by its' own styling and the H3 headline has no styles at all.

Important: global styles do NOT affect the styles of elements inside the Shadow DOM! 

<template id="custom-component">
    <style>
        h2 {
            color: lightcoral;
        }
    </style>
    <div class="customComponent">
        <h2>Inline template custom component</h2>
        <h3>Headline is (un)affected of global styling</h3>
        <p>This is only true for browsers supporting native Shadow DOM. If the headline above is green, your browser doesn't support Shadow DOM yet.</p>
    </div>
</template>
