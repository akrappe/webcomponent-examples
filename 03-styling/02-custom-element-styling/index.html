Custom Element styling - No Shadow DOM used here

Globally styled headline

The H2 and H3 headlines above are affected by the global styles defined in the head section.
The Web Components' H2 headline is affected by its' own styling and the H3 headline has no styles at all, so it falls back to global styling.

Important: this example does NOT use the Shadow DOM! 

<template id="custom-component">
    <style>
        .customComponent {
            display: inline-block;
            border: 1px solid red;
            padding: 10px;
        }
        .customComponent h2 {
            color: lightcoral;
        }
    </style>
    <div class="customComponent">
        <h2>H2 headline has own Custom Element styling</h2>
        <h3>H3 headline is affected by global styling</h3>
    </div>
</template>
